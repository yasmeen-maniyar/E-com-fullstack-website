# E-Commerce Website (Full Stack Project)

This is a full-stack e-commerce web application currently under development.

## Project Overview
The application simulates a real-world online shopping platform with user authentication, admin login, and database integration.

## Key Features
- User Signup & Login
- Admin Authentication
- REST API integration
- Frontend & Backend connectivity
- Secure database storage
- Protected routes
- Product management logic

## Project Status
Major core functionalities have been completed including authentication flow and database connection. Minor enhancements and UI improvements are currently in progress.
